Console Project

visual studio is used to creative launching pad that you can use to edit, debug, and build code, and then publish an app. to set it up you have to create a new file, add it into the area where it says file then go to open file and select your file. You then can download support for a certain code language you like. The console screen is used to provide user interface with little interaction. You need to use console.log() to print anything on console screen, and to go to the console screen, right click anywhere on the screen and tap on inspect, then you just look at the bottom of the screen that shows on the right and you'll see the console, or you can go to the top of it and tap the console button which will make it bigger.
